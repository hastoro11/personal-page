About Me

Hi There! I am Gabor Sornyei

I am 50 years old and a web developer based in Hungary, specializing in building dynamic web apps. I build clean, appealing, and functional interfaces which comply with the latest standards.

I graduated in 2016 in computer science and since then I have been working on web applications. I work both on front-end and back-and solutions. For the front-end technology I use React which is the latest and most modern standard in dynamic web applications. In the back-end I work with NodeJS and GraphQL and the database is provided by MongoDB.

In my works I try to follow the principle what Lenorado Da Vinci once laid out: "Simplicity is the ultimate sophistication."
